# baseConverter:

- A simple base converter class.
 
# Usage:

<code>app = BaseConverter()</code><br>
<code>result = app.convert(number, FromBase, ToBase)</code><br>
<code>print(result)</code>

# Example:

<code>app = BaseConverter()</code><br>
<code>result = app.convert(1519, 10, 2)</code><br>
<code>print(result)</code>
